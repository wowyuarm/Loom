# 任务状态

工程 skills 使用下列五种 triage 状态：

| 状态 | 含义 |
| --- | --- |
| `needs-triage` | 新想法或问题，尚未判断范围和处理方式。 |
| `needs-info` | 方向可能成立，但缺少关键信息。 |
| `ready-for-agent` | 描述和边界已足够清楚，可以交给 agent 实施。 |
| `ready-for-human` | 需要人做判断、拍板或亲自处理。 |
| `wontfix` | 已决定不做，并保留原因。 |

完成实施并验收后使用 `Status: resolved`；它是本地结案标记，不是 triage 状态。
